  # The Digital Craftsman: SOMA
  
  This is a code bundle for The Digital Craftsman's app which is SOMA (as part of my Final Year Project under Temasek Polytechnic, School of Design, CMD)
  SOMA is a node-based, modular creative application designed as a customisable toolbox for digital designers.

This project was built in response to contemporary creative ecosystems that prioritise automation, subscription models, and closed systems. Instead of fixed templates or paywalled features, SOMA runs locally and gives users full agency over how they construct their workflow.

The interface is structured around modular nodes that can be connected, rearranged, and expanded. Rather than prescribing outcomes, the system exposes structure and logic, allowing designers to build their own process architecture.

Users can also integrate their own code to extend functionality. The app is intentionally open, encouraging experimentation and modification instead of passive tool consumption.

SOMA is part of a larger research project, called The Digital Craftsman, exploring authorship, agency, and craft in the age of intelligent digital tools. It proposes a shift from operating within systems to consciously shaping them
  
  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
